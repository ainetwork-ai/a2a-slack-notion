  UNBLOCK MEDIA — A2A AGENT URLS (10 agents)
══════════════════════════════════════════════════════════════════════

Replace {BASE} with wherever you deploy unblock-agents, e.g.
  {BASE}=http://localhost:3000              (local dev)
  {BASE}=https://xyz.ngrok-free.app          (ngrok tunnel)
  {BASE}=https://unblock-agents.vercel.app   (Vercel deploy)

Every agent exposes:
  • GET  {BASE}/api/agents/{id}/.well-known/agent.json   → AgentCard
  • POST {BASE}/api/agents/{id}                          → JSON-RPC (A2A message/send, message/stream)
  • OPTIONS handled for CORS preflight (Access-Control-Allow-Origin: *)

══════════════════════════════════════════════════════════════════════
  CALLING A SKILL
══════════════════════════════════════════════════════════════════════

# Fetch an AgentCard:
curl {BASE}/api/agents/unblock-max/.well-known/agent.json

# Send a message to a skill (here: Max's "report" skill).
# metadata.variables fills the Notion template placeholders
# (^TODAY_DATE^, ^BASIC_ARTICLE_SOURCE^, ^CHIEF_COMMENT^, ...).
# Any placeholder not provided becomes "(제공되지 않음)" so the LLM
# doesn't echo the literal back in its answer.
curl -X POST {BASE}/api/agents/unblock-max \
  -H "Content-Type: application/json" \
  -d '{
    "jsonrpc": "2.0",
    "id": 1,
    "method": "message/send",
    "params": {
      "message": {
        "kind": "message",
        "messageId": "msg-1",
        "role": "user",
        "parts": [{ "kind": "text", "text": "비트코인 관점에서 최근 시장 분석해줘" }],
        "metadata": {
          "skillId": "report",
          "variables": {
            "TODAY_DATE": "2025-11-17",
            "BASIC_ARTICLE_SOURCE": "비트코인 현물 ETF 승인 후 ... (실제 원문)",
            "CHIEF_COMMENT": "이번 주 핵심 건으로 다뤄줘"
          }
        }
      }
    }
  }'

# Plain chat with the persona (no skillId):
curl -X POST {BASE}/api/agents/unblock-max \
  -H "Content-Type: application/json" \
  -d '{
    "jsonrpc": "2.0",
    "id": 1,
    "method": "message/send",
    "params": {
      "message": {
        "kind": "message",
        "messageId": "msg-1",
        "role": "user",
        "parts": [{ "kind": "text", "text": "안녕하세요!" }]
      }
    }
  }'

# Streaming response (Server-Sent Events):
curl -N -X POST {BASE}/api/agents/unblock-max \
  -H "Content-Type: application/json" \
  -H "Accept: text/event-stream" \
  -d '{ "jsonrpc":"2.0","id":1,"method":"message/stream","params":{ ... } }'
