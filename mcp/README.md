# MCP Integrations

This directory contains configuration for MCP (Model Context Protocol) integrations available in the Slack workspace.

## Available Providers

### Polymarket
- Trending prediction markets
- Market search by keyword
- Market details and odds

### News
- Search news articles by keyword
- Trending news topics
- Google News and Google Trends integration

## Usage

In any channel with MCP integrations enabled:

- `/polymarket trending` — Show trending prediction markets
- `/polymarket search <query>` — Search markets
- `/news search <query>` — Search news articles
- `/news trending` — Show trending news topics

## Channel Setup

Admins can enable/disable MCP integrations per channel via the Channel Detail Panel > Integrations tab.
